The squares below represent the themes available in mr-excel. By clicking on these squares, three files are generated: two Excel files with a negative and black & white text theme, and another file is an image with color, which includes a theme index.

You can also find the code related to the generated Excel file in the Selected Code section.

When clicking on a square, you may be prompted for permission to download multiple files simultaneously. This permission request occurs because we initiate the download of three files at the same time.
